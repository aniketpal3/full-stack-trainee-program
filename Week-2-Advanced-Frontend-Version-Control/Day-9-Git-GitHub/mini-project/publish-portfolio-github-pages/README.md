# Mini Project – Publish Portfolio Website on GitHub Pages

## Objective

Publish your portfolio website online using GitHub Pages.

---

# Project Used

Use:

```txt
Week 1 Day 1 + Day 2 Portfolio Website
```

Files:

```txt
index.html
about.html
projects.html
contact.html
style.css
```

---

# Step 1: Create Repository

Repository Name:

```txt
portfolio-site
```

Visibility:

```txt
Public
```

---

# Step 2: Upload Project

Inside Portfolio Folder:

```bash
git init

git add .

git commit -m "Portfolio website"

git branch -M main

git remote add origin REPOSITORY_URL

git push -u origin main
```

---

# Step 3: Enable GitHub Pages

Open Repository

Go to:

```txt
Settings
↓
Pages
```

Source:

```txt
Deploy from branch
```

Branch:

```txt
main
```

Folder:

```txt
/ (root)
```

Save.

---

# Step 4: Wait for Deployment

GitHub takes 1–5 minutes.

---

# Step 5: Open Website

Format:

```txt
https://username.github.io/portfolio-site/
```

Example:

```txt
https://palaniket227.github.io/portfolio-site/
```
