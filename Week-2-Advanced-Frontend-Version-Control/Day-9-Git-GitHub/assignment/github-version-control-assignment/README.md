# Day 9 Assignment – Git & GitHub Practice

## Objective

Practice GitHub repository management, branching, merging, and open-source exploration.

---

# Assignment 1

## Create Repository for Number Guessing Game

Use your Day 4 project.

Repository Name:

```txt
number-guessing-game
```

---

### Steps

Create Repository

```txt
number-guessing-game
```

Initialize Git

```bash
git init
```

Add Files

```bash
git add .
```

Commit

```bash
git commit -m "Initial commit"
```

Connect Repository

```bash
git remote add origin REPOSITORY_URL
```

Push

```bash
git branch -M main

git push -u origin main
```

# Assignment 2

## Create Feature Branch

Create:

```bash
git checkout -b feature-readme
```

Create:

```txt
README.md
```

Commit:

```bash
git add .

git commit -m "Add README file"
```

Merge:

```bash
git checkout main

git merge feature-readme
```

Push:

```bash
git push origin main
```
