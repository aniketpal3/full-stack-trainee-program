# Day 10 Assignment – Portfolio Enhancements

## Objective

Enhance the responsive portfolio website by adding extra interactive features.

---

## Assignment Features

### 1. Image Gallery Section

A gallery section is added with hover effects.

### 2. Phone Number Validation

Contact form includes phone number validation.

Rules:

- Phone number is required
- Phone number must be 10 digits
- Only numbers allowed

### 3. Sticky Navbar

Navbar stays fixed at the top while scrolling.

```css
position: sticky;
top: 0;
```

### 4. Dark Mode Toggle

A button toggles between light mode and dark mode.
