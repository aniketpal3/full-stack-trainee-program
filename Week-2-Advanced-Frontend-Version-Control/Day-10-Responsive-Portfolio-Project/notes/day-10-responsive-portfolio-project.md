# Day 10 – Mini Project: Responsive Website with Interactive JavaScript

## Learning Objectives

By the end of Day 10, students will:

- Combine HTML, CSS, and JavaScript in one complete project.
- Build a responsive personal portfolio website.
- Create navigation with smooth scrolling.
- Add dynamic projects using JavaScript.
- Validate contact form inputs.
- Add image gallery, sticky navbar, phone validation, and dark mode.
- Publish project using GitHub Pages.

---

## Project Overview

We will build a Responsive Personal Portfolio Website.

The website will include:

- Home section
- About section
- Skills section
- Projects section
- Image gallery
- Contact form
- Dark mode toggle
- Responsive navbar
- Smooth scrolling

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- GitHub Pages

---

## Main Features

### 1. Responsive Navbar

Navigation should work properly on:

- Desktop
- Tablet
- Mobile

---

### 2. Smooth Scrolling

Clicking navbar links smoothly scrolls to sections.

---

### 3. Dynamic Projects

Projects will be added using JavaScript array.

Example:

```js
const projects = ["Calculator App", "To-Do List", "Number Guessing Game"];
```

---

### 4. Contact Form Validation

Form fields:

- Name
- Email
- Phone
- Message

Validation:

- All fields required
- Email must contain `@`
- Phone number should be 10 digits
- Message should not be empty

---

### 5. Image Gallery

Gallery section with hover effects.

---

### 6. Dark Mode

Dark mode changes website theme using JavaScript.

---

## GitHub Pages Deployment

Steps:

```bash
git add .
git commit -m "Complete portfolio project"
git push origin main
```

Then enable GitHub Pages from repository settings.
