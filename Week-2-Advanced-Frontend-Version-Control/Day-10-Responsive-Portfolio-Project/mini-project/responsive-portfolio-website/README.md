# Responsive Portfolio Website

## Project Overview

This is a responsive personal portfolio website built using HTML, CSS, and JavaScript.

It combines all frontend concepts learned in Week 1 and Week 2.

---

## Features

- Responsive navigation bar
- Sticky navbar
- Smooth scrolling
- Home section
- About section
- Skills section
- Projects section
- Dynamic project rendering using JavaScript
- Gallery section with hover effects
- Contact form validation
- Phone number validation
- Dark mode toggle
- Responsive design for desktop, tablet, and mobile

## How to Run

1. Open this folder in VS Code.
2. Open `index.html`.
3. Right-click and choose **Open with Live Server**.
4. Test navigation, dark mode, projects, and contact form.

---

## Contact Form Validation Rules

- Name is required.
- Email is required.
- Email must contain `@` and `.`
- Phone number is required.
- Phone number must be 10 digits.
- Message is required.

## GitHub Pages Deployment

1. Push project to GitHub.
2. Open your repository on GitHub.
3. Go to **Settings**.
4. Open **Pages**.
5. Select source:

```txt
Deploy from branch
```

6. Select branch:

```txt
main
```

7. Select folder:

```txt
/ root
```

8. Click Save.
9. Wait 1–5 minutes.

Your live link format:

```txt
https://your-username.github.io/repository-name/
```
