# Day 6 Assignment – DOM Events Practice

## Objective

Practice DOM manipulation and JavaScript events by building small interactive features.

---

## Features Included

### 1. Counter App

- Increase counter using `+`
- Decrease counter using `-`

### 2. Image Gallery

- Click thumbnail image
- Show selected image in larger preview

### 3. Clear All List

- Add custom list items
- Clear all items using button

### 4. Live Character Counter

- Type text in textarea
- Count characters live
- Limit set to 280 characters

---

## Concepts Used

- DOM Selection
- getElementById()
- querySelectorAll()
- addEventListener()
- click event
- keyup event
- createElement()
- appendChild()
- innerText
- innerHTML
- style updates

1. Open this folder in VS Code.

## Learning Outcomes

After completing this assignment, students will understand:

- How to handle click events
- How to update UI dynamically
- How to create and remove elements
- How to build interactive frontend features
