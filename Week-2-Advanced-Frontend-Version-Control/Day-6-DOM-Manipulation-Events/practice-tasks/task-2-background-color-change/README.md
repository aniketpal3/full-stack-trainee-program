# Task 2 – Change Background Color

## Objective

Create a button that changes the page background color when clicked.

---

## Concepts Used

- DOM Selection
- addEventListener()
- click event
- Arrays
- Random Number Generation
- Style Manipulation

---

## Folder Structure

```txt
task-2-background-color-change/
├── index.html
├── script.js
└── README.md
```

---

## How It Works

1. User clicks button.
2. JavaScript selects a random color.
3. Background color changes.

---

## Expected Result

Every click changes the page background color.

---

## Learning Outcome

Students learn how to respond to user actions using events.
