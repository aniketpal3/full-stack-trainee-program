# Task 1 – Change Text Style

## Objective

Select a paragraph using its ID and change:

- Text
- Color
- Font Size

using JavaScript DOM manipulation.

---

## Concepts Used

- DOM
- getElementById()
- innerText
- style.color
- style.fontSize

---

## Folder Structure

```txt
task-1-change-text-style/
├── index.html
├── script.js
└── README.md
```

---

## Expected Output

Original:

```txt
This is the original paragraph.
```

Updated:

```txt
Paragraph updated using DOM Manipulation.
```

Color becomes blue and font size becomes 24px.

---

## Learning Outcome

Students learn how to select and update DOM elements dynamically.
