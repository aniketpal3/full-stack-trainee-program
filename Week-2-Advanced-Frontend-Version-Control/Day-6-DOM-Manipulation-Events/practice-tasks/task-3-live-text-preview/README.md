# Task 3 – Live Text Preview

## Objective

Create a text input box and display typed text in real-time below the input field.

---

## Concepts Used

- DOM Selection
- Input Events
- keyup event
- innerText
- Real-time UI update

---

## Folder Structure

```txt
task-3-live-text-preview/
├── index.html
├── script.js
└── README.md
```

---

## How It Works

1. User types in the input box.
2. The `keyup` event runs every time a key is released.
3. JavaScript reads the input value.
4. The typed text is displayed below the input box.

---

## Expected Output

If user types:

```txt
Hello DOM
```

Preview shows:

```txt
Hello DOM
```

---

## Learning Outcome

Students learn how to capture user input and update the webpage in real-time.
