# Task 4 – Add List Item

## Objective

Create a button that adds a new list item inside a `<ul>`.

---

## Concepts Used

- DOM Selection
- createElement()
- appendChild()
- click event
- Dynamic HTML creation

---

## Folder Structure

```txt
task-4-add-list-item/
├── index.html
├── script.js
└── README.md
```

---

## How It Works

1. User clicks the **Add Item** button.
2. JavaScript creates a new `<li>` element.
3. Text is added to the list item.
4. The list item is appended to the `<ul>`.

---

## Expected Output

Clicking the button creates:

```txt
New List Item 1
New List Item 2
New List Item 3
```

---

## Learning Outcome

Students learn how to dynamically create and add HTML elements using JavaScript.
