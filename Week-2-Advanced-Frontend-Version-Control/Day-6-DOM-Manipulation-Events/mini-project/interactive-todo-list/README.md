# Interactive To-Do List

## Project Overview

This is an interactive To-Do List built using HTML, CSS, and JavaScript DOM events.

Users can:

- Add a task
- Delete a task
- Mark a task as completed
- Clear all tasks

---

## Features

- Add tasks using button
- Add tasks using Enter key
- Delete individual task
- Toggle completed status
- Clear all tasks
- Empty list message
- Responsive design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Concepts Used

- DOM Manipulation
- getElementById()
- createElement()
- appendChild()
- addEventListener()
- click event
- keyup event
- classList.toggle()
- remove()
- innerHTML

## How to Run

1. Open folder in VS Code.
2. Open `index.html`.
3. Right-click and choose **Open with Live Server**.
4. Add, complete, delete, and clear tasks.

---

## Learning Outcomes

After completing this project, students will understand:

- How JavaScript interacts with HTML
- How to create elements dynamically
- How to handle user events
- How to update the UI without reloading page
