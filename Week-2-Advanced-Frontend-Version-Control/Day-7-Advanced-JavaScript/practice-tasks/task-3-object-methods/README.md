# Task 3 – Object Methods

## Objective

Create a JavaScript object named `book` with properties and a method.

---

## Object Properties

- title
- author
- pages

---

## Object Method

```js
details()
```

This method prints all information about the book.

---

## Concepts Used

- Objects
- Object Properties
- Object Methods
- this keyword
- Template Literals
