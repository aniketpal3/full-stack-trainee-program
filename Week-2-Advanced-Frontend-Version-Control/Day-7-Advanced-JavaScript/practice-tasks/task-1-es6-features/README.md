# Task 1 – ES6 Features

## Objective

Practice modern JavaScript ES6 features.

---

## Features Covered

### Arrow Functions

```js
const greet = () => {}
```

### Template Literals

```js
`Hello ${name}`
```

### Destructuring

```js
const { name } = person;
```

### Spread Operator

```js
const newArr = [...arr];
```
