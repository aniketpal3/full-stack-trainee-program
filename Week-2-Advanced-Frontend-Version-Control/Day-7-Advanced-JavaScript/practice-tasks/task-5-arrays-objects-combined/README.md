# Task 5 – Arrays and Objects Combined

## Objective

Create an array of employee objects and increase each employee salary by 10%.

---

## Employee Object Structure

```js
{
  name: "Rahul",
  salary: 30000
}
```

---

## Concepts Used

- Arrays
- Objects
- map()
- forEach()
- Template Literals
- Object Properties

---

## Folder Structure

```txt
task-5-arrays-objects-combined/

```

---

## Expected Output

```txt
Rahul - ₹33000
Aditi - ₹44000
Aman - ₹38500
```

---

## Learning Outcome

Students learn how to combine arrays and objects for real-world data handling.
