# Task 2 – Array Methods

## Objective

Practice JavaScript array operations.

---

## Tasks Performed

1. Create array of colors
2. Add a color using push()
3. Remove first color using shift()
4. Print all colors using forEach()

---

## Concepts Used

- Arrays
- push()
- shift()
- forEach()
