# Task 4 – Higher Order Array Methods

## Objective

Practice JavaScript higher-order array methods using marks.

---

## Given Array

```js
[50, 70, 40, 90, 30]
```

---

## Tasks

### filter()

Keep marks greater than or equal to 50.

### map()

Add 5 bonus marks to every mark.

### reduce()

Find total marks.

---

## Concepts Used

- Arrays
- filter()
- map()
- reduce()
- Arrow Functions
- join()
