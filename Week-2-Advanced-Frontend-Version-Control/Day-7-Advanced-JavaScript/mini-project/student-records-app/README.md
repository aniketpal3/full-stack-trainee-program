# Student Records App

## Project Overview

Student Records App is a simple JavaScript project built using arrays, objects, ES6 features, and higher-order array methods.

Users can:

- View all students
- Add a new student
- View toppers
- Calculate class average

---

## Features

- Display all student records
- Add new student using form input
- Filter toppers with marks greater than or equal to 80
- Calculate class average using reduce()
- Responsive UI
- DOM-based rendering

---

## How to Run

1. Open this folder in VS Code.
2. Open `index.html`.
3. Right-click and choose **Open with Live Server**.
4. Add new students and check records.
