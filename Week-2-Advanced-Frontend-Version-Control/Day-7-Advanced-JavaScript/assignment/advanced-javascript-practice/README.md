# Day 7 Assignment – Advanced JavaScript Practice

## Objective

Practice ES6 features, arrays, objects, and higher-order array methods.

---

## Assignment Tasks

### 1. Even Numbers using filter()

Create an array of numbers and return only even numbers.

### 2. Laptop Object

Build an object `Laptop` with:

- brand
- ram
- price
- details() method

### 3. Marks Analysis

Write a program that takes an array of marks and prints:

- Highest marks
- Lowest marks
- Average marks

### 4. Add Student Function

Extend the Student Records App by adding a function:

```js
addStudent(name, marks)
```

---
