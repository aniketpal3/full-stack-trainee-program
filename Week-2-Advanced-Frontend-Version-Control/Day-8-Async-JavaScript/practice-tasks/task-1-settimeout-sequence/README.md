# Task 1 – setTimeout Sequence

## Objective

Print messages asynchronously using setTimeout.

---

## Task

Print:

```txt
One
Two
Three
```

with delays.

---

## Concepts Used

- setTimeout()
- Async JavaScript
- Event Loop

---

## Folder Structure

```txt

```

---

## Expected Output

```txt
One

(after 1 second)

Two

(after 2 more seconds)

Three
```

---

## Learning Outcome

Students understand asynchronous execution using setTimeout.
