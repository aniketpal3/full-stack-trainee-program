# Task 2 – Callback Function

## Objective

Understand how callbacks work.

---

## Task

Create:

```js
doHomework(subject, callback)
```

Function should:

1. Print homework completed.
2. Execute callback function.

---

## Concepts Used

- Functions
- Parameters
- Callback Functions


## Learning Outcome

Students understand how callbacks allow one function to execute another function after completion.
