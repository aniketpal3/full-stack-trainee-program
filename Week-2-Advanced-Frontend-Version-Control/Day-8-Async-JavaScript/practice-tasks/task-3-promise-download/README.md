# Task 3 – Promise Download Complete

## Objective

Create a Promise that resolves after 3 seconds with the message:

```txt
Download Complete
```

---

## Concepts Used

- Promise
- resolve()
- then()
- setTimeout()
- DOM Manipulation

---

## Folder Structure

```txt
task-3-promise-download/
├── index.html
├── script.js
└── README.md
```

---

## How It Works

1. Page first shows `Download not started`.
2. JavaScript starts a Promise.
3. Status changes to `Downloading file...`.
4. After 3 seconds, Promise resolves.
5. Output becomes `Download Complete`.

---

## Expected Output

```txt
Downloading file...

After 3 seconds:

Download Complete
```

---

## Learning Outcome

Students learn how Promises handle asynchronous tasks.
