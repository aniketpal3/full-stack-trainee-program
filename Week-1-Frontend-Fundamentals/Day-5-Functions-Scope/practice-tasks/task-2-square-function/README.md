# Task 2 – Square Function

## Objective

Create a function:

```js
square(n)
```

that returns the square of a number.

---

## Concepts Used

- Functions
- Parameters
- Return Values
- Variables
- DOM Manipulation

---

## Files Included

```txt
task-2-square-function/
├── index.html
├── script.js
└── README.md
```

---

## How It Works

The function receives a number.

Example:

```js
square(6)
```

Returns:

```txt
36
```

---

## Expected Output

```txt
Square of 6 = 36
```

---

Students learn:

- Function Parameters
- Return Statements
- Mathematical Operations
