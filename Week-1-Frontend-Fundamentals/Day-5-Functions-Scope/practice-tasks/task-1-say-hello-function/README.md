# Task 1 – Say Hello Function

## Objective

Create a function named `sayHello()` that prints:

```txt
Hello Student
```

three times.

---

## Concepts Used

- Functions
- Function Calling
- Return Statement
- DOM Manipulation
- Console Output

---

## Files Included

```txt
task-1-say-hello-function/
├── index.html
├── script.js
└── README.md
```

---

## How It Works

The function returns:

```txt
Hello Student
```

The function is called three times.

---

## Expected Output

```txt
Hello Student
Hello Student
Hello Student
```

---

## Learning Outcome

Students learn how to:

- Create functions
- Call functions
- Reuse code
