# Simple To-Do List

## Project Overview

This is a simple To-Do List application built using HTML, CSS, and JavaScript.

Users can:

- Add tasks
- Mark tasks as completed
- Delete tasks

This project focuses on JavaScript functions and basic DOM manipulation.

---

## Features

- Add new task
- Show all tasks
- Mark task as completed
- Delete task
- Empty task validation
- Responsive design

## Main Functions

### addTaskFromInput()

Gets task from input field and validates it.

### addTask(task)

Adds a new task object into the tasks array.

### showTasks()

Displays all tasks on the page.

### deleteTask(index)

Deletes task based on index.

### markTaskCompleted(index)

Marks selected task as completed.

## How to Run

1. Open this folder in VS Code.
2. Open `index.html`.
3. Right-click and choose **Open with Live Server**.
4. Add tasks and test complete/delete features.
