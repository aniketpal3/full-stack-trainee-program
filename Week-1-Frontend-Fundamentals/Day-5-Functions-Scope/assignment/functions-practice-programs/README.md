# Day 5 Assignment – Functions Practice Programs

## Objective

The objective of this assignment is to practice JavaScript functions in depth using real programming problems.

---

## Programs Included

### 1. Factorial Function

Create a function:

```js
factorial(n)
```

that returns the factorial of a number.

Example:

```txt
5! = 120
```

---

### 2. Reverse String Function

Create a function:

```js
reverseString(str)
```

that returns the reversed string.

Example:

```txt
JavaScript → tpircSavaJ
```

---

### 3. Palindrome Function

Create a function:

```js
isPalindrome(str)
```

that checks if a string reads the same backward.

Example:

```txt
madam → palindrome
```

---

### 4. Mark Task as Completed

Extend the To-Do List logic by adding a function to mark a task as completed.
