# Mini-Project: Styled Portfolio Website

## 📌 Objective

Convert your Day 1 HTML portfolio into a professional, responsive, and beautifully styled website using CSS.

## ⏱️ Estimated Time

**60-90 minutes**

## 📝 Project Description

A complete 4-page portfolio website with professional styling, responsive design, and modern UI components.

## 🎨 Design Highlights

### Color Scheme

```css
--primary: #667eea (Bright Blue)
--secondary: #764ba2 (Purple)
--dark: #2c3e50 (Dark Gray)
--light: #ecf0f1 (Light Gray)
--white: #ffffff (White)
```

### Typography

- **Headings:** Segoe UI, Bold
- **Body:** Segoe UI, Regular
- **Size Hierarchy:** 40px (h1) → 32px (h2) → 22px (h3)

### Layout

- **Desktop:** 1200px max-width container
- **Flexbox:** Navigation, hero, contact form
- **CSS Grid:** Skills, projects grid
- **Gap Spacing:** 30px between items

---

## 📱 Responsive Breakpoints

| Device | Width | Columns | Layout |
|---|---|---|---|
| **Mobile** | < 480px | 1 | Single column, stacked |
| **Tablet** | 481-768px | 2 | Two columns |
| **Desktop** | 769px+ | 3 | Three columns |
