# 🎯 Number Guessing Game

## Project Overview

A simple JavaScript game where the computer randomly selects a number between 1 and 10.

The player must guess the number.

The game gives hints:

- Too High
- Too Low
- Correct

The player gets only 5 attempts.

---

## Features

✅ Random Number Generation

✅ User Input Validation

✅ Too High / Too Low Messages

✅ Win Message

✅ Maximum 5 Attempts

✅ Game Over Message

✅ Reset Game Option
