# Day 4 Assignment – JavaScript Control Structures

## Objective

Practice JavaScript control structures using real programming problems.

---

## Programs Included

### Program 1 – Multiplication Table

Generate multiplication table from 1 to 10.

Example:

```txt
7 × 1 = 7
7 × 2 = 14
7 × 3 = 21
```

---

### Program 2 – Prime Number Check

Check whether a number is prime or not.

Example:

```txt
17 is a Prime Number
```

---

### Program 3 – Sum from 1 to 100

Calculate total sum using a loop.

Example:

```txt
Sum = 5050
```

---

- Loops

- DOM Manipulation
