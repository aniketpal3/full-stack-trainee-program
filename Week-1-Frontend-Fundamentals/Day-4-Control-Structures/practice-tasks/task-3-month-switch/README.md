# Task 3 – Month Name Using Switch

## Objective

The objective of this task is to print the month name based on a month number using the JavaScript `switch` statement.

For example:

- 1 means January
- 2 means February
- 3 means March
- 12 means December

---

## Concepts Used

- JavaScript variables
- switch statement
- case
- break
- default
- DOM manipulation
- console output
