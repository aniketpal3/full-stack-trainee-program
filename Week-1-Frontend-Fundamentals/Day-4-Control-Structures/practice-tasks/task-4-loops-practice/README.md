# Task 4 – Loops Practice

## Objective

The objective of this task is to practice JavaScript loops.

This task includes:

1. Printing numbers 1 to 10 using a `for` loop.
2. Printing even numbers from 1 to 20 using a `while` loop.
3. Printing array elements using a `for...of` loop.

---

## Concepts Used

- for loop
- while loop
- for...of loop
- if condition
- arrays
- modulus operator
- DOM manipulation
- console output
