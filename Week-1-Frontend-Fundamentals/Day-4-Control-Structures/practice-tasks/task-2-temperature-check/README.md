# Task 2 – Temperature Check

## Objective

The objective of this task is to check temperature using JavaScript conditions.

The program displays:

- `Hot` if temperature is above 30
- `Warm` if temperature is between 15 and 30
- `Cold` if temperature is below 15

---

## Concepts Used

- JavaScript variables
- if condition
- else if condition
- else condition
- comparison operators
- logical AND operator
- DOM manipulation
