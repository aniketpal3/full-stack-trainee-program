# Task Manager Enhancements

## Objective

Extend the Task Manager App with advanced features.

---

## Assignment Features

### 1. Edit Task Title

Users can update task title using the Edit button.

---

### 2. Task Categories

Each task can have a category:

```txt
Work
Personal
Urgent
General
```

---

### 3. Sort Tasks by Creation Date

Latest tasks appear first using:

```js
.sort({ createdAt: -1 })
```

---

### 4. Multiple Users

Tasks include a `userId` field so tasks can be linked to users.

---

## Learning Outcomes

Students learn:

- Full stack CRUD
- MongoDB schema design
- Frontend-backend integration
- Task filtering and organization
- Real-world project enhancement
