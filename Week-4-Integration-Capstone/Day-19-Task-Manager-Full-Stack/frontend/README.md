# Task Manager Frontend

## Project Overview

Frontend UI for Task Manager App.

Users can:

- Add Tasks
- View Tasks
- Complete Tasks
- Edit Tasks
- Delete Tasks

---

## Technologies Used

- HTML
- CSS
- JavaScript
- Axios

---

## Features

### Add Task

```txt
Create new task
```

### View Tasks

```txt
Fetch from backend
```

### Complete Task

```txt
Mark completed
```

### Edit Task

```txt
Update title
```

### Delete Task

```txt
Remove task
```

### Categories

```txt
Work

Personal

Urgent

General
```
