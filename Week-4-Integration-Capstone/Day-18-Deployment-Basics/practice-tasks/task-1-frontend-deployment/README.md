# Practice Task 1

## Objective

Deploy your Day 10 Portfolio Project.

---

## Option 1 – GitHub Pages

### Step 1

Push project to GitHub.

```bash
git add .

git commit -m "Portfolio Project"

git push origin main
```

---

### Step 2

Open repository.

Go to:

```txt
Settings
↓
Pages
```

---

### Step 3

Select:

```txt
Branch: main

Folder: /root
```

Click:

```txt
Save
```

---

### Step 4

Wait 2–5 minutes.

Website becomes live:

```txt
https://username.github.io/repository-name
```

---

## Option 2 – Netlify

### Step 1

Create account:

https://netlify.com

---

### Step 2

Click:

```txt
Add New Site
```

---

### Step 3

Import GitHub Repository

or

Drag & Drop project folder.

---

### Step 4

Deploy

Netlify generates URL:

```txt
https://random-name.netlify.app
```

---

## Deliverable

Share:

```txt
Live URL

GitHub Repository URL
```

---

## Learning Outcome

Students learn:

- Frontend deployment
- GitHub Pages
- Netlify Hosting
