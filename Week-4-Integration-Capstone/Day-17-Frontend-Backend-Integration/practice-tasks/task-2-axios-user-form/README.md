# Task 2 – Axios User Form

## Objective

Create a frontend form and send user data to backend using Axios.

## Backend Required

```txt
POST http://localhost:5000/users
```

## Run

1. Start backend server.
2. Open `index.html` with Live Server.
3. Fill form and submit.

## Concepts Used

- Axios
- POST request
- Form handling
- DOM events
- Error handling
