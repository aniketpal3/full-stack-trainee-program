# Frontend Auth Dashboard

## Objective

Create frontend signup, login, profile, and logout functionality connected to backend Auth API.

## Features

- Signup form
- Login form
- JWT token saved in localStorage
- Protected profile fetch
- Logout button
- Error handling

## Backend Required

Run Day 16 mini project:

```txt
http://localhost:5000
```

Required routes:

```txt
POST /signup
POST /login
GET /profile
```

## How to Run

1. Start backend server.
2. Open `index.html` with Live Server.
3. Signup.
4. Login.
5. Fetch profile.
6. Logout.

## Concepts Used

- Axios
- localStorage
- JWT
- Protected route
- Frontend-backend integration
