# User Dashboard Full Stack

## Project Overview

This project connects frontend HTML, CSS, and JavaScript with a backend User CRUD API.

The dashboard can add, view, search, update, and delete users.

## Features

- Display users in a table
- Add new user
- Edit user
- Delete user
- Search user by name
- Axios API calls
- Responsive dashboard UI

## Backend Required

Use Day 15 mini project:

```txt
Week-3-Backend-Development/Day-15-Mongoose-CRUD-Operations/mini-project/user-crud-api-mongodb
```

Backend base URL:

```txt
http://localhost:5000
```

Required routes:

```txt
GET /users
POST /users
PUT /users/:id
DELETE /users/:id
```

## How to Run

### Step 1: Start Backend

```bash
cd Week-3-Backend-Development/Day-15-Mongoose-CRUD-Operations/mini-project/user-crud-api-mongodb
npm install
node server.js
```

### Step 2: Open Frontend

Open:

```txt
index.html
```

with Live Server.
