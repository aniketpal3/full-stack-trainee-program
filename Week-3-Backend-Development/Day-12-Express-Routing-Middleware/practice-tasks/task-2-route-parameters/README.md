# Task 2 – Route Parameters

## Objective

Learn how to use route parameters in Express.

---

## Install Dependencies

```bash
npm install
```

---

## Run Server

```bash
node server.js
```

---

## Test Routes

Home:

```txt
http://localhost:3001
```

Route Parameter:

```txt
http://localhost:3001/hello/Rahul
```

Output:

```txt
Hello Rahul
```

Example:

```txt
http://localhost:3001/hello/Aniket
```

Output:

```txt
Hello Aniket
```

---

## Concepts Used

- Express Routing
- Route Parameters
- req.params

---

## Learning Outcome

Students learn dynamic routing using route parameters.
