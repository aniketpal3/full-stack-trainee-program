# Task 1 – Basic Express Server

## Objective

Create and run your first Express.js server.

---

## Install Dependencies

```bash
npm install
```

or

```bash
npm install express
```

---

## Run Server

```bash
node server.js
```

---

## Open Browser

```txt
http://localhost:3000
```

---

## Expected Output

```txt
Hello from Express.js!
```

---

## Concepts Used

- Express
- Routing
- GET Request
- Server Setup

---

## Learning Outcome

Students learn how to create and run an Express server.
