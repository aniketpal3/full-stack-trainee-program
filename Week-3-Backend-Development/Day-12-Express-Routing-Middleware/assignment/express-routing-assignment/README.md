# Express Routing Assignment

## Features

- /weather/:city
- GET /api/users
- POST /api/users
- Static Contact Form

## Install

```bash
npm install
```

## Run

```bash
node server.js
```

## Routes

```txt
http://localhost:4000/weather/delhi
```

```txt
http://localhost:4000/api/users
```

```txt
http://localhost:4000/contact.html
```
