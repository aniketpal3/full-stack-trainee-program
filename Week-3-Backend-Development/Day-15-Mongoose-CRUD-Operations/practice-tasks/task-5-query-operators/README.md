# Task 5 – Query Operators

## Objective

Use MongoDB query operators with Mongoose.

---

## Practice Tasks

- Get all users aged between 20 and 30.
- Get all users whose name is either Aman or Neha.

---

## Install

```bash
npm install
```

---

## Run

```bash
node server.js
```

---

## Query Operators Used

```js
$gte
$lte
$or
```

---

## Database Used

```txt
day15-crud-db
```

---

## Important Note

Run Task 1 first to create users.

---

## Learning Outcome

Students learn how to filter data using MongoDB operators.
