# Task 3 – Update Users

## Objective

Update user data using Mongoose.

---

## Practice Tasks

- Update a user's email address.
- Increase age of all users by +1.

---

## Install

```bash
npm install
```

---

## Run

```bash
node server.js
```

---

## Database Used

```txt
day15-crud-db
```

---

## Concepts Used

- findOneAndUpdate()
- save()
- find()
- async/await
- update operations

---

## Important Note

Run Task 1 first to create users.

## Learning Outcome

Students learn how to update single and multiple MongoDB documents.
