# Task 4 – Delete Users

## Objective

Delete documents from MongoDB using Mongoose.

---

## Practice Tasks

- Delete one user by ID.
- Delete all users older than 60.

---

## Install

```bash
npm install
```

---

## Run

```bash
node server.js
```

---

## Database Used

```txt
day15-crud-db
```

---

## Concepts Used

- findByIdAndDelete()
- deleteMany()
- findOne()
- query filters

---

## Important Note

Run Task 1 first to create users.

---

## Learning Outcome

Students learn how to remove documents from MongoDB.
