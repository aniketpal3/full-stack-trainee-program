# Task 2 – User Schema & Model

## Objective

Create a Mongoose Schema and Model.

---

## Install

```bash
npm install
```

---

## Run

```bash
node server.js
```

---

## User Schema

```js
{
  name: String,
  email: String,
  age: Number
}
```

---

## Concepts Used

- Mongoose
- Schema
- Model
- MongoDB

---

## Learning Outcome

Students learn how MongoDB collections are represented using Models.
