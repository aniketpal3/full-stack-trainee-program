# Task 1 – MongoDB Connection

## Objective

Connect Node.js with MongoDB using Mongoose.

---

## Install

```bash
npm install
```

---

## Run

```bash
node server.js
```

---

## Expected Output

```txt
MongoDB Connected
```

or

```txt
MongoDB Atlas Connected
```

---

## Concepts Used

- MongoDB
- Mongoose
- Database Connection

---

## Learning Outcome

Students learn how to connect Node.js with MongoDB.
