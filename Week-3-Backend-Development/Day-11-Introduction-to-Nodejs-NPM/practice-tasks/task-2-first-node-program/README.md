# Task 2 – First Node.js Program

## Objective

Create and run your first Node.js program.

---

## File

```txt
hello.js
```

---

## Code

```js
const today = new Date();

console.log("Name: Aniket Pal");

console.log("Today's Date:");

console.log(today.toDateString());

console.log("Current Time:");

console.log(today.toLocaleTimeString());
```

---

## Run Program

```bash
node hello.js
```

---

## Example Output

```txt
Name: Aniket Pal

Today's Date:

Sat Jun 14 2026

Current Time:

11:20:45 AM
```

---

## Concepts Used

- console.log()
- Date Object
- Node Runtime

---

## Learning Outcome

Students learn:

- Execute JavaScript outside browser
- Run Node.js files
- Use built-in JavaScript objects in Node.js
