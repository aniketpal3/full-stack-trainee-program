# Task 1 – Install Node.js & Verify Setup

## Objective

Install Node.js and verify installation.

---

## Step 1

Download Node.js

Official Website:

https://nodejs.org

Download:

```txt
LTS Version
```

---

## Step 2

Install Node.js.

Follow installer instructions.

---

## Step 3

Open Terminal.

Verify Node.js:

```bash
node -v
```

Example:

```txt
v22.17.0
```

---

## Step 4

Verify npm:

```bash
npm -v
```

Example:

```txt
10.9.2
```

---

## Commands Used

```bash
node -v

npm -v
```

---

## Expected Result

Node.js and npm installed successfully.

---

## Learning Outcome

Students learn how to install and verify Node.js environment.
